# 🧠 QuizHub — Платформа для квизов с рейтингом

SPA-приложение на Vanilla JS с Firebase-бэкендом и Google-аутентификацией. Проходи квизы на русском или английском, соревнуйся с друзьями и занимай место в таблице лидеров!

## 🚀 Демо
[httpsтвой-логин.github.ioquizhub](httpsтвой-логин.github.ioquizhub)

## 🛠 Стек
- Frontend Vanilla JS (ES6+), Bootstrap 5.3, CSS Animations
- Backend Firebase (Auth + Firestore)
- API Open Trivia DB (английские вопросы)
- Локальная база 52 русских вопроса в 8 категориях
- Деплой GitHub Pages

## ✨ Функциональность
- ✅ Google-авторизация — вход через Google-аккаунт
- ✅ Выбор языка — 🇷🇺 Русский (локальная база)  🇬🇧 English (Open Trivia DB API)
- ✅ 10 категорий — общие знания, наука, история, спорт, кино, музыка, география, IT
- ✅ 3 уровня сложности — легкосреднесложно (влияет на множитель очков)
- ✅ Таймер — 15 секунд на вопрос, красный при 5 сек
- ✅ Система очков — бонус за скорость + множитель сложности
- ✅ Таблица лидеров — топ-3 с медалями 🥇🥈🥉 + полная таблица
- ✅ Real-time обновления — таблица лидеров обновляется без перезагрузки
- ✅ Конфетти — при результате 70+ очков
- ✅ Тост-уведомления — обратная связь на все действия
- ✅ Неоновый дизайн — геймерская тема с плавающими частицами
- ✅ Адаптив — корректно работает на мобильных устройствах
- ✅ Офлайн-режим — русские вопросы работают без интернета

## 🔧 Локальный запуск
1. Склонируй репозиторий
2. Замени `firebaseConfig` в `js/firebase-config.js` на свой (см. ниже)
3. Запусти через Live Server (VS Code) или `python -m http.server 8000`
4. Открой `http://localhost:8000`

## 🔑 Firebase-конфигурация
Для работы нужен Firebase-проект с включёнными сервисами:
1. [Создай проект](https://console.firebase.google.com) → `quizhub`
2. **Authentication** → Sign-in method → **Google** (включить)
3. **Firestore Database** → Создать → **Тестовый режим** → `eur3`
4. Настройки проекта → Добавить приложение → Веб → скопировать `firebaseConfig`
5. Вставить конфиг в `js/firebase-config.js`
